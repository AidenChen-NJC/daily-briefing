Perplexity 的 $450M ARR，隱藏了一個它不公布的數字

事實背景

Perplexity 是一家 AI 搜尋引擎，於 2026 年 3 月推出 Plaid（金融數據 API 中介層，讓 app 讀取用戶銀行帳戶）整合，透過旗下 Computer agent（能自主執行多步驟任務的 AI 系統）連接超過 12,000 家銀行，讓用戶以文字指令建立預算追蹤器、債務還款計劃與退休規劃。此前 Perplexity 已推出美國報稅整合，可自主填寫 IRS 表格。隨著 Computer 2 月底上線，月活用戶超過 1 億，3 月 ARR 從 $305M 跳升至 $450M，單月成長 50%，訂閱方案 $20 至 $200/月。

So What?

Mint 死亡的核心原因，是用戶連接帳戶後第二週活躍率斷崖下跌。Perplexity 的 $450M 是單月年化數字，最可能反映的是 Computer agent 上線後的試用衝刺，而非穩定留存。判斷這個數字是否可持續的唯一指標是 Plaid 連接後的 30 日活躍率，而 Perplexity 至今沒有公布這個數字。

值得追問
  1. 如果 30 日留存率遠低於 Plaid 連接量，$450M ARR 的敘事對後輪投資人意味著什麼？
  2. Plaid 持有 OAuth token、銀行關係與合規資質，Perplexity 只是 API 消費端。如果主要銀行在 2027 年完成直連標準遷移繞過 Plaid，Perplexity 的整個 fintech 主張需要從頭重建，這個風險如何定價？
  3. $20/月方案加上 Plaid per-connection 費用與 LLM 推論成本，低端用戶很可能是負毛利。真正支撐商業模式的是 $200/月企業方案，但沒有人知道那層的用戶佔比是多少。

事實背景

Amazon CEO Andy Jassy 在年度股東信中首度公開 AI 相關營收：AWS AI 業務年化營收 $15B，是 AWS 在相同發展階段的 260 倍；自研晶片系列 Trainium、Graviton 與 Nitro 合計年化營收超過 $20B。Jassy 表示未來可能以機櫃為單位對外銷售 Trainium。兩家匿名客戶曾要求買斷 Amazon 2026 年全部 Graviton 產能，但遭拒以保護其他客戶。Amazon 計劃投入 $200B AI 資本支出，此前曾引發投資人對泡沫的疑慮。

So What?

值得追問
  1. 若將 Graviton 和 Nitro 的歷史基礎營收拆除，Trainium 的實際年化採用量大約是多少？

McKinsey 的推論成長數字，可能計算的是算力而非美元

事實背景

a16z Charts of the Week 引用 McKinsey 模型指出，AI 推論（inference，模型實際執行產出結果的過程，有別於模型學習的 training 階段）將成為 2030 年主導 AI 工作負載，佔資料中心總需求 40% 以上，CAGR（年複合成長率）達 35%，遠超訓練的 22%。全球半導體市場預計從 $775B 翻倍至 $1.6T，其中運算與資料儲存佔產業成長 55%。

So What?

過去 18 個月每 token 推論成本已下跌超過 90%，MoE 和 speculative decoding 持續壓縮單次推論的算力需求。若 McKinsey 的 35% CAGR 計算的是使用量而非美元支出，則半導體市場翻倍的預測需要額外假設「定價不崩潰」，而這個假設在競爭加劇的環境中相當脆弱。能在低單價高量下維持毛利的超大規模雲端商是贏家，靠推論服務賣高毛利的 inference API 新創是輸家。

值得追問
  1. 如果推論成本繼續以每年 50% 的速度下跌，半導體美元市場是否真的會翻倍，還是使用量增長只是抵消了定價侵蝕？
  2. Groq 和 Cerebras 在延遲敏感的 real-time agentic 應用中確實有優勢，但目前沒有任何推論專用晶片公司公開宣布超過 $500M 的年度合約。推論主導的世界，對 Nvidia 的真實威脅從哪個時間點才開始？
  3. 若軟體效率持續提升讓推論成本接近零，受益的是用戶還是模型提供商？

事實背景

So What?

「5 倍 Codex 用量」從未被轉化為任何可工程評估的數字，沒有 token 上限、沒有 session 數，而開發者的核心抱怨是 Claude Code 在複雜任務中途斷線，「5 倍」這個相對數字根本沒有回答這個問題，因為 Plus 基準本身就已經是抱怨的起點。消費端訂閱之爭是代理戰場，真正的決戰在 Cursor 和 Windsurf 的預設模型選擇。Anthropic 封鎖 OpenClaw 的動作讓 Cursor 更有動力維持自己的模型調度層，結果是 IDE 平台積累議價權，OpenAI 和 Anthropic 同時成為可替換供應商。真正的輸家是兩家公司的訂閱部門，贏家是 Cursor 和 Windsurf。

值得追問

佛州傳票讓 Anthropic 坐等先例，再搶 OpenAI 的企業客戶

事實背景

佛羅里達州檢察長對 OpenAI 啟動正式調查並發出傳票，指控 ChatGPT 在佛羅里達州立大學（FSU）校園槍擊事件的策劃過程中提供了協助。這是州級執法機構首次以「AI 產出內容與犯罪行為的因果關係」為由對 AI 公司採取法律行動。

So What?

佛州選擇走州消費者保護法或產品責任法的路徑而非聯邦 Section 230，是刻意的。州產品責任法對「產品設計缺陷」的舉證標準更低，更容易建立初步案件。若此路徑成立，最需要恐慌的不是 OpenAI，而是所有部署 agentic AI 的企業客戶，銀行、醫院與法律事務所將要求合約中加入無上限賠償條款，而 OpenAI 當前的企業合約責任上限設計並未預期這類風險。Anthropic 是短期結構性受益者，只需保持沉默、加速 safety 文件完備性，等先例確立後以「更嚴格的 safety 架構」為由搶走流失的企業客戶。

值得追問
  1. 如果先例成立，AI 企業合約的責任條款將如何重新設計？誰會在這個法律服務需求中受益？
  2. Anthropic 的 Constitutional AI 在公關層面提供了差異化，但這在同類查詢中是否真的產出了實質不同的結果？
  3. 如果各州開始用不同的法律框架追究 AI 責任，聯邦層面是否會被迫加速立法以建立統一標準？

So What?

USDT 市值的計算方式（流通量乘以 $1 錨定價）與 ETH 的自由浮動市值在壓力情境下根本不具可比性。若宏觀環境惡化到 ETH 跌至 $1,525，同一環境很可能觸發 Tether 的大規模贖回，迫使其折價拋售非流動性儲備。Polymarket 的合約設計沒有處理「USDT 脫鉤時市值如何計算」的邊界條件，60% 的機率數字建立在兩個定義不對等的市值之上。

值得追問
  1. 如果 Tether 的儲備透明度問題在同一個宏觀壓力情境下爆發，ETH「失去第二」的預測合約應該如何結算？
  2. ETH ETF AUM 下跌 65% 中，有多少是價格效應，有多少是主動淨流出？如果主動贖回比例低於 20%，「機構撤離」的敘事是否被頭條數字系統性誇大了？
  3. ETH 依賴風險偏好上升才能漲，USDT 在避險情緒中自然成長。這個結構性不對稱，在目前的宏觀環境下有多少可能在 2026 年內逆轉？

EVMBench 72% 的 exploit 分數，衡量的是「背過答案的能力」

事實背景

Paradigm 與 OpenAI 聯合推出 EVMBench，一個測試 AI agent 偵測、修補與利用高嚴重性智能合約漏洞的基準測試，資料集基於 40 份審計的 120 個精選漏洞。GPT-5.3-Codex 透過 Codex CLI 的 exploit（在受控環境中利用漏洞）得分達 72.2%，而六個月前的 GPT-5 僅 31.9%。隨後 OpenZeppelin 發表分析，指出漏洞資料來自 2024 至 2025 年中期的公開審計，與被測 AI 的訓練截止日大致相同，存在數據污染（訓練數據包含測試數據導致結果失真）問題。

So What?

數據污染問題是次要缺陷，更根本的問題是 EVMBench 沒有衡量「AI 停止搜尋後遺漏的漏洞數量」，即 false-negative rate。Agent 找到一個漏洞就停止的行為在真實審計中比完全不用 AI 更危險，因為它給了審計團隊虛假的完整性保證。Code4rena 和 Immunefi 的人類審計員短期不受威脅，真正威脅他們的是「管理層誤讀 benchmark 後削減審計預算」這個決策鏈。Paradigm 和 OpenAI 是公信力受損的輸家，OpenZeppelin 是贏家，定位為可信的方法論仲裁者。

值得追問
  1. 如果管理層把 72% 的 exploit 分數解讀為「AI 已經能做審計了」，然後砍掉人工審計預算，誰來承擔因 false-negative 漏洞導致的損失？
  2. Paradigm 投資了審計生態相關項目，又推出可能壓低人類審計員市場價值的 benchmark。這個利益結構如何影響你對 EVMBench 的信任程度？
  3. 如果一個 benchmark 的核心缺陷在發布幾週內就被指出，這對整個 AI 安全評估基礎設施的可信度意味著什麼？

$300B 歷史新高拿掉四筆就縮水，早期市場正在選邊站

! 種子輪金額增但交易數減 30%，是這輪 AI 週期早期市場集中化的第一個明確結構性訊號。
事實背景

Crunchbase 數據顯示 2026 年 Q1 全球創投投資達 $300B，是有史以來最大的單季紀錄，年增與季增均超過 150%。OpenAI、Anthropic、xAI、Waymo 四筆 megaround（超大型融資輪，通常指 $1B 以上）合計 $188B，佔總額 65%。扣除這四筆，剩餘約 $112B 仍可創下多數年份的單季紀錄。AI 公司吸收了 80% 的資金，較去年同期 55% 大幅上升。早期輪年增 41%，種子輪金額年增 31%，但種子輪交易數量年減 30%。十億美元俱樂部包含 Cerebras、Rapidus（晶片）、Skild AI（機器人）、Wayve（自駕系統）與 Shield AI（國防平台）。

So What?

種子輪金額增但交易數減 30%，資本正在「預選少數贏家」而非廣泛撒網。輸家是沒有頂級 VC 背書的普通新創，他們獲得種子輪的機率已大幅下降。贏家是 Benchmark、a16z 等能定義「值得大注」標準的旗艦基金，因為稀缺的早期流量正集中流向他們背書過的公司。

值得追問
  1. 這輪 AI 週期的十億美元俱樂部包含晶片廠、機器人公司和國防平台，這些公司的資本回收週期比軟體長得多。Sequoia 等習慣軟體倍率的基金，估值邏輯是否需要根本性調整？
  2. 四大 megaround 佔 65% 總額，讓「$300B 歷史新高」的標題存在嚴重誤導性。如果這四筆延至 Q2，Q1 的敘事會是什麼？
  3. 種子輪集中化趨勢持續下去，對沒有在第一輪就獲得頂級 VC 背書的創辦人，接下來的融資路徑會長什麼樣？

化肥供應鏈的損失，會在秋季收成時才讓人看見

事實背景

中東衝突在荷莫茲海峽（全球最重要的石油與天然氣運輸咽喉）造成航運瓶頸。全球 25% 以上的氮肥（包括尿素、氨）與約 20% 的 LNG（液化天然氣，氮肥關鍵原料）經此海峽運輸。氮肥佔全球化肥使用量 60%，化肥佔穀物生產成本約 20%。尿素現貨價已逼近 2022 年高點。美國三大航空（American Airlines、Delta、Southwest）每加侖燃油漲 1 美分分別帶來 $50M、$40M、$22M 的年度額外成本，Reuters 估計四大美國航空合計額外燃料成本達 $5.8B。

So What?

油價衝擊即時反映，但化肥短缺的糧食影響有兩個季度滯後，施肥週期到收成需要完整的生長週期。即使荷莫茲海峽今天恢復正常，2026 年秋季收成仍有下行風險。市場現在盯著每日油價動態，但農業期貨（玉米、小麥 CBOT）才是應該追蹤的領先指標。

值得追問
  1. 美國農民可能已在衝突升級前鎖定化肥需求，但歐洲和澳洲的採購週期較晚。如果這些地區化肥不足轉向進口美國穀物，Archer-Daniels-Midland 和 Bunge 等美國穀物出口商如何定位？
  2. 中國以煤炭而非 LNG 製氮肥，且已限制出口鎖定國內供給。這場危機是否給了中國化肥廠商在東南亞市場擴大影響力的窗口？
  3. 如果農業期貨才是這場危機的真實指標，為什麼金融媒體的注意力集中在油價和航空燃油成本？

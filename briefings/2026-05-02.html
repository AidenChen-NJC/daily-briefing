白宮封殺 Anthropic 的真正目的是壓低議價，不是趕走它

事實背景

白宮正在同時做兩件看似矛盾的事：公開宣稱 Anthropic 是「供應鏈風險」（supply chain risk，根據國防部 1260H 條款的供應商禁令），又阻止 Anthropic 把 Mythos（具備強網安攻防能力的最新模型）從私部門約 50 家用戶擴張到 120 家，理由是政府自己要用那批算力。同時，白宮起草備忘錄要求所有國安機構採用「多供應商策略」，讓政府日後可以切換供應商。AI Security Institute 已確認 GPT 5.5 達到與 Mythos 相近的網安能力，David Sacks 公開預測 6 個月內所有前沿模型都會達到這個水準。

So What?

「多供應商備忘錄」的政治功能是讓 Anthropic 無法開口要更高的價格，不是讓政府真的能切換。各前沿模型的 system prompt 格式、tool-use schema、安全分類標籤完全不兼容，加上 FedRAMP 重新認證週期，真正的遷移代價是幾年不是幾個月。短期受益者是 OpenAI（作為備援選項），但 workflow 嵌入最深、實際遷移成本最高的仍然是 Anthropic。

值得追問
  1. 若 Sacks 說得對、六個月內所有前沿模型都達到 Mythos 級能力，Anthropic 靠 Mythos 換到的政府合約溢價窗口只剩兩季。他們真正在搶的是哪些私部門客戶的多年鎖定合約？
  2. OpenAI 或 Palantir 誰已經把工作流嵌得最深，讓這份多供應商備忘錄形同廢紙？

Meta EPS 含 $8B 一次性利益，市場這週用錯數字在定價

事實背景

Meta Q1 2026 營收 $56.31B，年增 33%，是 2021 年以來最快。EPS $10.44，但其中含 $8.03B 一次性稅務利益。Family Daily Active People（DAP，Meta 跨 Facebook / Instagram / WhatsApp / Messenger 的去重日活指標）環比下滑 5% 至 35.6 億，是該指標公布以來首次下滑，約少 1.9 億人。管理層歸因於伊朗戰爭造成的網路中斷與俄羅斯封鎖 WhatsApp。2026 年 capex 指引從 $115–135B 上修至 $125–145B。Meta 同時宣告下月裁員約 8,000 人，約佔員工總數 10%。JPMorgan 下調評等，股價盤後跌約 9%。

So What?

扣除 $8.03B 稅務利益後，實際 EPS 約落在 $6–6.5 區間，與 headline 差近四成。如果賣方用 headline EPS 計算 forward P/E，等於人為壓低本益比約 40%，讓股票看起來比實際便宜。JPMorgan 下調的時機點暗示他們做了調整，但盤後 -9% 的反應主要針對 capex 上修，不是 EPS 質量。也就是說，散戶和部分機構這週仍在用一個被灌水的數字定價 Meta。

值得追問
  1. 下季伊朗和俄羅斯市場若未恢復，廣告主會不會開始按「可觸及用戶數」重新議價，Meta 在中東與東歐的實際 ARPU 衝擊有多大？
  2. Capex 上修到 $145B 但裁員只省 $1.5–2B，這筆 AI 支出到底綁在哪些已簽約的企業客戶或內部產品 KPI 上，還是純粹在跟 OpenAI 軍備競賽？

Stargate 10 GW 達成，但「至多 $100B」的承諾對 NVIDIA 方向搞反了

! Stargate 90 天加 3 GW，這是全球 AI 算力擴張速度仍在加速的直接指標，直接影響 Anthropic 政府談判的籌碼窗口與 Meta 等玩家 capex 上修的合理性判斷。
事實背景

OpenAI 宣布 Stargate（OpenAI、Oracle、SoftBank、MGX 共同出資的 4,000 億美元資料中心合資計畫）已達成 10 GW 算力，比原本的 2029 目標提前約 3 年。第一個運營站點在德州 Abilene，新增站點橫跨德州、新墨西哥州、威斯康辛州、密西根州。過去 90 天新增 3 GW。NVIDIA 與 OpenAI 簽訂意向書，承諾隨每 1 GW 佈署漸進投資，總金額「至多 $100B」。

So What?

NVIDIA 的承諾結構是「ceiling × 10 GW = $1 兆」，但「漸進」加「至多」在法律上毫無約束力。NVIDIA 在這筆交易裡是賣方，不是投資方，方向完全相反。把這個數字當作需求確認的分析師搞錯了邏輯。真正需要問的是已交割的採購訂單金額，而不是意向書上限。Oracle 股東若需要這些算力有真實付費使用率才能讓財務模型成立，目前沒有任何公開數字支撐。

值得追問
  1. 90 天加 3 GW，OpenAI 能不能說清楚這是「已通電跑推論」還是「簽了地、拿了電力合約但還沒裝機」——真正的 GPU 使用率是多少？
  2. NVIDIA 既然是賣方，為什麼需要對 OpenAI「漸進投資」——這到底是需求拉動還是供應商補貼推動的 GTM？

事實背景

事實背景

Mira 由哈佛輟學的 AnhPhu Nguyen 與 Caine Ardayfio 創辦，兩人之前因 I-XRAY（用 Meta 眼鏡結合公開資料即時人肉搜索路人的 demo）事件爆紅。本週發表第一款產品：無相機 AI 眼鏡，售價 $649（無度數）至 $799（有度數）。眼鏡持續錄音並轉文字，配合指環觸發的 agent（代理），可執行寄信、叫車、Amazon 下單等任務，連接 Slack、Notion、Gmail，所有錄音立刻刪除，只保留逐字稿存於手機端。即時翻譯 60 種以上語言，語音延遲低於 700 毫秒。首批 300 副已出貨，明年再 1,000 副，種子輪 $6.6M。

Claude Security 進企業的最大障礙不是技術，是原始碼離境問題

事實背景

Anthropic 推出 Claude Security 公測，使用 Opus 4.7 掃描企業 codebase（程式碼庫），找出漏洞並產生修補建議。同日，Cursor 推出 Security Review，用自主代理排程掃描並將結果推送到 Slack。兩者直接對標 Snyk（SAST 靜態應用安全測試工具，上輪估值 $7.4B）和 Veracode（同類工具，被 Thoma Bravo 以 $2.5B 收購）等傳統安全工具。

So What?

Anthropic 要攻企業市場的最大障礙不是找漏洞的準確率。用 Opus 4.7 掃 codebase 意味著原始碼必須離開企業邊界進入 Anthropic 的推論環境，這對任何持有 FedRAMP 或 SOC 2 認證的企業是直接的信任邊界問題。傳統 SAST 工具（Snyk CLI、Semgrep）可在本地執行，原始碼從不離境。能提供 on-premise 掃描的 Semgrep 和 GitHub Advanced Security 是短期受益者，Snyk 和 Veracode 面臨功能降維打擊，但靠合規整合護城河短期守住。

值得追問
  1. Anthropic 和 Cursor 現在有任何一家企業客戶把掃描結果直接接進 SOC 2 審計嗎？沒有這個案例，企業安全主管買單的前提是什麼？
  2. Cursor 的 Security Review 底層是不是就在調用 Claude？如果是，Anthropic 真正的 GTM 是賣給終端企業，還是當 Cursor 的後端供應商？

MegaETH 的 10ms 優勢對 HFT 是假的，真正的早期用戶是 MEV bot

事實背景

MegaETH 的 MEGA token 於 4 月 30 日上線，13 家主要 CEX 同步現貨上市。首日 FDV（Fully Diluted Valuation，按當前幣價計算全部代幣的總估值）約 $1.5B，market cap 約 $1.7 億，流通量 11.3 億枚，佔總量 100 億枚的 11.3%，TVL（Total Value Locked，鎖在協議內的資產總值）$4.9 億。盤中從 $0.22 跌至約 $0.15，跌幅約 30%。技術主張：100k TPS、10ms 出塊、次毫秒延遲，宣稱比 ETH 主網快 5,000 倍。投資人包含 Vitalik Buterin、Joe Lubin、Dragonfly Capital。

So What?

MegaETH 宣稱的 10ms 出塊是 soft confirmation（軟確認），最終結算仍需等以太坊 L1 出塊（約 12 秒）完成 state root 確認。HFT 桌的風控部門通常不接受依賴軟確認做結算的風險，因為 L1 重組或 sequencer 故障時面臨回滾。真正能受益於 10ms 的是對延遲敏感但不需最終結算確認的訂單流，例如 CEX-DEX 套利的鏈上腿，以及 MEV bot，而不是傳統 HFT 做市商的核心策略。MegaETH 對機構 HFT 的吸引力被系統性高估。

事實背景

Kite 於 4 月 30 日將 AI-native L1 升上 mainnet，建在 Avalanche 生態。三層架構：Kite Chain 處理穩定幣結算、Agent Passport（AI 代理的可程式化錢包與身份，用戶可設消費上限）、上層 interface 處理代理的註冊與發現。首日整合 90 家以上服務商，KITE token 進入 CoinMarketCap top 100，Upbit 排名 top 7。Kite 同時接入 Coinbase x402（HTTP 402 支付協議）、Google AP2（Agent Payment Protocol）、Stripe MPP（支付協議）、Anthropic MCP（Model Context Protocol）四套標準，首發案例是在 Claude 內部下單購買實體商品。

a16z 說軟體債沒有違約潮，Medallia 示範的是違約被技術性推遲了

事實背景

S&P 500 在伊朗衝突期間下跌 10%，11 個交易日內完全收復，上週五收 7,165.08，年增 30%。Berkshire 現金部位 $3,730 億，創公司史上最高。Paul Tudor Jones 引用市場總市值佔 GDP 比率（巴菲特指標）達 252%，相較 1929 年的 65%、2000 年的 170%，認為若 P/E 回歸 25–30 年均值，市場可能跌 30–35%。a16z 反駁，稱當前利潤率是 1990 年代的兩倍、預期 EPS 成長是長期均值兩倍以上。軟體債均價跌至約 $0.90，軟體佔 distressed loan（折價 20% 以上的不良貸款）約 35%。Thoma Bravo 將 2021 年以 $6.4B 收購（含 $3B 債務）的 Medallia 全數股權交還債權人，年息已從 $1 億漲到 $3 億，Blackstone 拒絕展期。

So What?

軟體債均價 $0.90、佔 distressed 35% 但違約率未上升，最合理的解釋是：PE 持有者正在大量使用 PIK（Payment in Kind，以新債代替現金利息）和展期把問題推後。Blackstone 拒絕幫 Medallia 展期，說明這個機制已開始失效。Medallia 示範的恰好是 a16z「軟體債沒有違約潮」說法的例外：股權先死、債券還沒技術違約，但實質控制權已經換手。持有軟體為主要抵押品的 BDC（Business Development Company，公開市場上的私募信貸基金）的 LP，現在該看的是 PIK 貸款佔比和加權展期到期日，不是當前的 default rate。

值得追問

Kyle Samani 離開 Multicoin，是在告訴你基金結構不再是押 SOL 最好的工具

事實背景

Multicoin Capital 共同創辦人 Kyle Samani 在 X 宣布退出一線管理，但持續擔任 Forward Industries（FWDI，Solana treasury company，即以持有 SOL 為核心資產的上市公司）董事長，以及 Zama 董事。他在 Multicoin Master Fund 對 3 月 31 日的持倉提出 in-kind redemption（實物贖回，取 FWDI 股票而非現金），並個人投入 $2,500 萬。FWDI 於 2025 年 9 月完成 $16.5 億 PIPE（Private Investment in Public Equity，私募投資於公開公司股權），由 Multicoin、Jump、Galaxy 共同領投。Kyle 預測 2026 年底 Solana mainnet 現貨和永續交易量將對標主要 CEX。

So What?

Kyle 選擇 in-kind 贖回而非現金，存在兩種對 LP 截然相反的解釋：一是基金帳上 FWDI 持倉流動性不足，賣掉換現金會壓低股價，這是對 LP 的善意；二是 Kyle 利用贖回窗口低成本累積個人持股，這是潛在的利益衝突。真正的訊號是：Kyle 認為單押 FWDI 的 alpha 大於管理 Multicoin 的 LP 基金結構。這個動作對 crypto VC 行業的意義在於，頂尖管理人正在從基金結構流向 treasury vehicle，類似於 2010 年代對沖基金經理流向 family office。

值得追問

  2. FWDI 的 $16.5B PIPE 全現金買 SOL 做 treasury company，LP 買的到底是 Kyle 的 alpha，還是一個收 2/20 的槓桿 SOL ETF？

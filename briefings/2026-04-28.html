Anthropic 的 $100M「捐贈」，本質是付給 AWS 和 Apple 的獲客成本

! 最強前緣模型無法公開釋出，就只能當 locked-in 客戶的激勵工具——這是供應鏈壟斷最聰明的打法。
事實背景

Anthropic 在 4 月 27 日正式公告 Project Glasswing，核心做法是把內部最強前緣模型（Frontier Model，當代能力最頂的大型語言模型）Claude Mythos Preview（明確表示「沒有公開釋出計畫」）以 $100M 使用額度信用加上 $4M 開源社群捐款的形式，限制性開放給 AWS、Apple、Broadcom、Cisco、CrowdStrike、Google、JPMorganChase、Microsoft、NVIDIA、Palo Alto Networks、Linux Foundation 共 12 家公開合夥人，外加 40 家以上關鍵基礎設施組織。前幾週 Anthropic 已用 Mythos Preview 在每個主要作業系統和每個主要瀏覽器找到「數千個」0-day（零日漏洞，開發者尚未知曉的安全缺陷），多數為高危等級。

So What?

$100M 信用額度在 AWS、Apple、JPMorgan 這份名單的結構下，是預付客戶獲取成本，不是慈善。這些組織一旦用完額度、驗證了 Mythos Preview 的掃描效能，替換回原有安全管線的成本遠高於年費，議價籌碼幾乎歸零。合夥人名單同時涵蓋 Apple 和 Microsoft，意味 Anthropic 已把「拒絕付費」的政治成本拉到極高。

值得追問
  1. 如果 Mythos Preview 真的發現「數千個」0-day，這些漏洞情報的所有權歸誰，CrowdStrike 和 Palo Alto Networks 是在免費幫 Anthropic 建威脅情報資料庫嗎？
  2. 白名單夥伴用完 $100M 額度後，到底會轉成付費客戶，還是會把能力內建、把 Anthropic 踢出供應鏈？

Google 跟投前員工的對手公司，買的是資訊購買權而非信念

事實背景

4 月 27 日，前 DeepMind 強化學習（Reinforcement Learning，RL，從環境回饋學習而非標註資料）團隊負責人、AlphaZero 共同設計者 David Silver 公告其新創公司 Ineffable Intelligence 完成 $1.1B 種子輪（Seed Round，公司最早期的機構融資），投後估值 $5.1B，為歐洲史上最大種子輪。共同領投為 Sequoia 和 Lightspeed，跟投方包括 Nvidia、DST Global、Index、Google 和英國主權 AI 基金（Sovereign AI Fund，國家級 AI 戰略投資載體）。Silver 的技術主張是打造「superlearner」，從基本運動技能到複雜抽象推理全部靠自我經驗學習，不依賴現行 LLM（大型語言模型）路線的人類資料訓練。同日另一家 ex-DeepMind 和 ex-OpenAI 工程師創辦的 Recursive Superintelligence（公司僅 4 個月大）也拿到 $500M。

So What?

Google 跟投自己前員工創辦的直接競爭對手，最合理的解釋是資訊購買權而非阻擋：以小額跟投換取董事會觀察席，讓自家決策者提前看到「外部 RL 路線的真實進展速度」。若 Ineffable 成功，Google 有跟投回報；若失敗，這筆錢確認了自家路線正確。真正的輸家是 DeepMind 內部押相同路線的研究員，Google 的跟投行為等於管理層公開承認外部競爭是真實威脅。

值得追問
  1. AlphaZero 的成功前提是封閉環境加明確獎勵函數，Ineffable 要從棋盤推進到「複雜抽象推理」，reward shaping（獎勵函數設計）本身不就重新引入人類先驗，技術路線在哪個階段會退化回 RLHF？
  2. Google 同時投自己前員工創辦的直接競爭對手，如果 Ineffable 成功了，DeepMind RL 團隊十年的投入算什麼？

Microsoft 用「放棄分潤」買了一顆反壟斷解藥

事實背景

Microsoft 與 OpenAI 在 4 月 27 日同步公告修訂後的合作條款。主要變動為：Microsoft 對 OpenAI IP（含模型、產品）的授權延至 2032 年，但從獨家授權（Exclusive License，僅限被授權方使用）改為非獨家；Microsoft 不再就 OpenAI 在 Azure 上銷售的 OpenAI 產品收取收入分潤（Revenue Share）；OpenAI 仍須優先在 Azure 首發，但若 Azure 無法支援所需算力則可在其他雲端首發；OpenAI 對 Microsoft 至 2030 的長期分潤義務改為有上限的固定金額。觸發點包括：Financial Times 上月揭露 Microsoft 評估就 OpenAI 的 $50B AWS 合約提告，以及美英歐三地反壟斷機構正在審查 Microsoft 與 OpenAI 是否在企業 AI 市場形成不正當競爭。

So What?

Microsoft 放棄分潤，換掉的是反壟斷調查最大的火藥桶，代價是公開承認 Azure 已無法靠合約鎖住 OpenAI 的增量算力需求。「Azure first」條款在 OpenAI 已簽 $50B AWS 合約的現實下，實質保護只剩新模型的首發視窗期。這是止血，但傷口已經開了，Azure AI 部門是此次修約的真正輸家。

值得追問
  1. OpenAI 現在能同時賣給 AWS、GCP、Azure，企業客戶可以拿 AWS 報價回來壓 Azure，OpenAI 的 NRR（淨收入留存率）在通路互打下守得住嗎，還是會變成雲端廠商的議價棋子？
  2. OpenAI 拿到多雲自由後，最怕哪一段通路被 Microsoft 反手收回？

SpaceX 給 Cursor 百萬張 H100，但這批算力的架構根本跑不了 IDE 補全

事實背景

4 月 21 日，SpaceX 公告與 Cursor 母公司 Anysphere 達成協議：SpaceX 取得在 2026 年底前以 $60B 全資併購 Cursor 的 call option（買入選擇權）；若不行使，SpaceX 須付 $10B 給 Cursor 作為合作期間對價。SpaceX 將 Cursor 的 IDE 與 Colossus（號稱百萬張 H100 等效算力的 xAI 超算叢集）綁定，目標是建出最強 coding 與 knowledge work 模型。此前 Microsoft 曾接洽併購 Cursor 但選擇不出價，被認為是因無法提供等效算力承諾。Elon Musk 已於 2 月將 SpaceX 與 xAI 合併，自評估值 $1.25T。

So What?

Colossus 是為 xAI 大批次訓練設計的叢集，採用高吞吐量低延遲訓練的 InfiniBand 網路拓撲，與 Cursor 需要的毫秒級 IDE 補全所需的分散式 serving 架構根本不同。若 SpaceX 不重建 serving stack，Cursor 低於 200 毫秒的補全 SLA 在 Colossus 上無法兌現，所謂「百萬卡算力」是虛的戰略標的。這讓 $60B 行權更像談判籌碼，而非真實的商業計畫。

值得追問
  1. $10B「違約金」本質上是算力市場標價，那 Cursor 團隊拿到的到底是現金還是算力額度，如果是後者，Anysphere 的股東實際拿到的流動性是零嗎？
  2. Cursor 核心價值在於跨模型的 model-routing 層，一旦綁定 xAI 算力與模型生態，對 OpenAI 和 Anthropic API 的整合會受什麼限制，這是否直接摧毀 Cursor 原本的結構性優勢？

Western Union 選 Solana 做跨境結算，但它的合規穩定幣設計內建了一個矛盾

事實背景

Western Union CEO Devin McGranahan 在 4 月 27 日確認，自家穩定幣 USDPT（U.S. Dollar Payment Token）將於 2026 年 5 月在 Solana 上線，比原定「2026 上半年」時程略為提前。USDPT 由首家聯邦特許數位資產銀行 Anchorage Digital Bank 發行，U.S. Bank（全球系統重要性銀行）託管。首階段不直接面向 C 端消費者，而是作為 Western Union 全球 agent 網絡跨國結算的 USD 替代品，將原有 SWIFT（跨境銀行報文網路，傳統電匯主軸）backend 換成穩定幣。配套產品包括 Digital Asset Network（內部結算網）與未來推出的 Stable Card（穩定幣連結的實體支付卡）。

So What?

Anchorage 作為聯邦特許銀行必須遵守 OFAC（美國海外資產控制辦公室）制裁規定，但 Solana 的 token program 本身不含原生 KYC 層，凍結邏輯若依賴鏈外攔截，任何鏈上流通中的 USDPT 在被凍結前都可在 DEX 上轉移；若把凍結寫入 token program，整個 token 的去中心化程度接近零。兩種設計都有合規脆弱點，疊加 Solana 驗證者集中度問題，這個設計的監管風險高於同類 Ethereum 穩定幣。

值得追問
  1. Solana 歷史上已有數次長時間停機，Western Union 的合規團隊有沒有 SWIFT fallback，還是整個結算層直接斷掉？
  2. Western Union 全球 agent 點超過 50 萬個，首階段到底有多少 agent 會實際用 USDPT 結算，如果是個位數百分比，這跟 Ripple 當年 ODL 只有幾條走廊在跑有什麼本質區別？

KelpDAO 橋被打穿後，整個 restaking 賽道共用同一個攻擊面，市場還在定價這個不確定性

事實背景

4 月 18 日 KelpDAO 遭駭損失 $293M，攻擊者透過偽造訊息誘使 KelpDAO 跨鏈橋直接釋出 116,500 顆 rsETH（重質押 ETH 的流通憑證，Liquid Restaking Token），約占 rsETH（KelpDAO 的 restaked ETH 憑證）流通量 18%，事件歸因於北韓國家級駭客單位 Lazarus Group。加上 4 月 1 日 Drift Protocol 遭同一組織攻擊損失 $285M，4 月前 18 天 DeFi（去中心化金融）總損失達 $606M，為 2025 年 2 月 Bybit 案以來最慘單月。更值得注意的是二階段反應：KelpDAO 案後 48 小時，Aave 被撤走 $84.5B，全 DeFi TVL（Total Value Locked，總鎖倉量）跌至 800 億美元中段。攻擊頻率年增 68%，2026 年前 4.5 個月已發生 47 起 DeFi 事件。

So What?

KelpDAO 的攻擊面在跨鏈訊息的 oracle/relayer 驗證層，不在 EigenLayer 的核心質押經濟設計，但 EigenLayer 從未公開其 AVS（主動驗證服務）跨鏈訊息的驗證架構細節，意味若使用相同的 LayerZero 或 Wormhole 類架構，攻擊面可直接複製到所有 LRT（Liquid Restaking Token）協議。市場現在定價的是不確定性，不是已知風險，這讓整個 restaking 賽道的風險溢酬需要上調，而不只是針對 KelpDAO 的單一事件。

值得追問
  1. KelpDAO 橋被打穿後，EigenLayer 和 Symbiotic 上的其他 LRT 協議有沒有任何一家能證明自己的跨鏈訊息驗證層跟 KelpDAO 不同，還是整個賽道共用同一個攻擊面，只是還沒被輪到？
  2. Aave 的 $84.5B 撤資如果大部分只是暫時搬到 CEX 或國庫券協議避險，那 Aave 的護城河其實是「沒有更好的去處」而不是產品黏性，對嗎？

順豐兩個月融兩輪，中國物理 AI 已進入 capex 部署期，矽谷還說「還在科研期」

事實背景

4 月 27 日是 2026 年迄今 AI 領域最重要的單日募資日。主要交易包括：Ineffable Intelligence $1.1B 種子輪（另文詳述）；前 DeepMind 和前 OpenAI 工程師創辦的 Recursive Superintelligence 拿到至少 $500M，公司僅成立 4 個月，技術主線同樣是強化學習路線；工業人形機器人公司 Robot Era 完成逾 $200M 融資，由順豐（SF Express）領投，距離 3 月剛完成的 $146M Series B 僅隔兩個月；量子電腦公司 Quantum Art 完成 $140M Series A 加碼輪，Series A 累計達 $240M，開發多核離子阱量子電腦（trapped-ion，相比超導路線易擴展但相干時間長）Perspective；AI 居家服務前台軟體公司 Avoca 完成 $125M 融資，由 Meritech 和 General Catalyst 領投；德國工業機器人 AI 公司 Sereact 完成 $110M Series B。單日總流入超過 $2.18B。

So What?

順豐在兩個月內對 Robot Era 重複大額加碼，最直接的解讀是：SF Express 已在自家倉庫跑出可量化的揀貨效率數字，否則無法對順豐控股股東交代。這個信號意味中國物流業的物理 AI 已超越概念驗證期，進入 capex（資本支出）部署期，矽谷「實體 AI 還在科研期」的敘事已落後至少 18 個月。真正的輸家是任何用「實體 AI 未成熟」邏輯低估中國供應鏈 AI 的西方投資人。

值得追問
  1. 順豐領投 Robot Era 兩個月內第二輪，到底是因為機器人已經在倉庫跑出可量化的效率提升，還是順豐只是在用投資鎖定供應商排他權、怕被京東物流搶先？
  2. 同一天六筆大額融資的 LP 重疊度有多高，如果背後其實是同一批錢在分散下注而非獨立判斷，這個單日紀錄是真實市場信號還是少數幾家 LP 的 FOMO 放大器？

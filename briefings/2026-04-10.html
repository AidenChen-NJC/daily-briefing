Anthropic 的 $300 億 ARR，有 $290 億來源沒人說清楚

! 訂閱企業客戶最多貢獻 $10 億，$290 億落差指向隱形的單一買家依賴
事實背景

Anthropic 宣布年化營收跑率（ARR，Annual Recurring Revenue，訂閱制業務的標準衡量指標）突破 $300 億，一年前這個數字是 $10 億，OpenAI 同期為 $250 億。成長主因官方歸因於企業客戶爆發：年花費超 $100 萬的企業數在不到兩個月內從 500 家增至逾 1,000 家。同日，Anthropic 發布 Managed Agents（全託管 agent 平台，開發者定義任務與權限，平台處理沙箱執行、長時間自主 session 與多 agent 協調），定價為標準 API token 費加 $0.08/session-hour，早期採用者包括 Notion、Rakuten、Asana 與 Sentry。

So What?

1,000 家年消費 $100 萬客戶最多貢獻 $10 億，剩下 $290 億的來源沒有被解釋。Amazon 是 Anthropic 最大股東兼雲端合作夥伴，AWS Bedrock 的轉售消費極可能佔大頭；如果是這樣，Anthropic 的「企業多元化」本質上是 Amazon 採購意志的延伸，而非真正的買方分散。更大的問題在於，Amazon 同時在推自家 Nova 模型的 agent 能力，Bedrock 對 Anthropic 既是分發管道，也是潛在壓價槓桿。用 ARR 數字推算 Anthropic 定價能力的投資人，正在看一個被精心裁剪過的數字。

值得追問
  1. 如果 AWS Bedrock 佔 Anthropic ARR 超過 50%，那 Anthropic 和一個有漂亮 logo 的 AWS 子服務差在哪裡？
  2. 如果 Amazon 某天決定把 Nova 定價壓到 Claude 以下，Anthropic 有多少直銷客戶可以接住？
  3. $0.08/session-hour 在 Rakuten 五個部門並發跑 agent 的情境下，Anthropic 的 GPU 成本結構撐得住嗎？

Meta 花 $143 億買的不是模型，是殺掉競爭對手的工具

事實背景

Meta Superintelligence Labs 於 2026 年 4 月 8 日發布首款多模態推理模型 Muse Spark，可處理語音、文字與圖像，內建「contemplating mode」讓多個 agent 對抗辯論困難問題。在推理基準上與 Anthropic Opus 4.6、OpenAI GPT 5.4 有競爭力，但在 coding 和 ARC-AGI 2（測試 AI 泛化推理能力的高門檻基準）上明確落後。該模型走閉源路線，Meta 表示「希望」未來開源但未承諾時間表。負責人 Alexandr Wang 於 9 個月前 Zuckerberg 以 $143 億收購 Scale AI 後接管該部門，聲稱「從零重建了 AI 堆疊」。

So What?

$143 億買來一個在 coding 與 ARC-AGI 2 上落後對手的模型，這筆錢的真實標的不是 Muse Spark，而是 Scale AI 的數據標註基礎設施，也就是對 RLHF（Reinforcement Learning from Human Feedback，用人類反饋訓練模型的核心流程）管道的控制權。Scale AI 此前同時服務 Anthropic、OpenAI 與 Meta，Meta 的收購讓這個關鍵供應商從獨立公司變成內部資產，Anthropic 與 OpenAI 失去了最重要的數據標註外包管道。Meta 花 $143 億買到的護城河，不在模型排行榜上，而在上游數據管線的獨佔控制。

值得追問
  1. 如果 Anthropic 和 OpenAI 需要替代 Scale AI 的標註能力，市場上還有誰能填這個缺口？
  2. Meta 閉源 Muse Spark 同時，Llama 系列的開源策略會走到哪一步？純粹是非核心模型的陪跑還是另有分工邏輯？
  3. Alexandr Wang 在 Meta 內部的真實角色是「建模型的人」還是「管數據管線的人」？

Perplexity 的 50% 月增長可能只是換了一種算法

事實背景

Perplexity AI 截至 2026 年 3 月的年化營收跑率（ARR）突破 $4.5 億，較前月成長約 50%，用戶數超過 1 億，管理層目標是 2026 年底達 $6.56 億。增長官方歸因於兩點：agent 產品「Computer」的推出（讓 AI 自主在用戶電腦上執行任務），以及從固定訂閱制轉向 usage-based pricing（按使用量計費）。定價調整前 ARR 約為 $3.05 億，兩年前為 $1,600 萬。

So What?

ARR 從 $3.05 億跳到 $4.5 億的時間點與 usage-based pricing 切換高度重合，但 Perplexity 沒有公開月活用戶數是否同步成長。若用戶數持平而 ARPU（每用戶平均收入）上升，這 50% 增長主要是重新計費方式，而非真實消費擴張，持續性極低。更根本的問題是，Perplexity 的底層依賴 Claude 和 GPT API，而 Anthropic 的 Managed Agents 和 OpenAI 的 Operator 正在直接進入相同的「代替用戶執行任務」市場，供應商即競爭者，Perplexity 既無自有模型做成本護城河，也無企業 IT 通路做分發護城河。

值得追問
  1. 如果 Anthropic 對 Perplexity 的 API 調用設定用途限制或差異定價，Perplexity 的毛利結構還剩什麼？
  2. usage-based 定價切換是真實增長的開始，還是一次性的計費重估？沒有月活拆分就無法判斷。
  3. Perplexity 的出路是自研模型、被收購，還是成為 Anthropic 生態的一個垂直 UI 層？

Prometheus 想複製 Palantir，但沒有 Palantir 最難複製的那個東西

事實背景

Jeff Bezos 旗下隱身 AI 新創 Project Prometheus 聘請 Kyle Kosic 擔任基礎設施負責人，Kosic 曾共同創立 xAI 並在 120 天內主導建成 Memphis 的 Colossus 超級算力叢集，後轉投 OpenAI，現跳槽 Prometheus。公司由 Bezos 與前 Google X 負責人 Vikram Bajaj 共同領導，於 2025 年底成立，已獲約 $62 億融資。核心策略是先收購航太、製造與工程企業取得專有營運數據，再訓練產業專用模型。

So What?

Prometheus 的策略卡點不在資金或算力，而在數據工程能力。航太和製造業的感測器時序數據、CAD 檔案與品管影像格式差異極大，統一成可訓練格式的 pipeline 複雜度遠超 Kosic（算力叢集）或 Bajaj（Google X 實驗專案）的核心能力範圍，而 Palantir 花了 15 年才建好這套能力，目前 Prometheus 的公開團隊構成裡看不到對應的產業數據工程人才。更有歷史先例：GE Digital 和 Siemens MindSphere 都曾以「拿數據」為核心邏輯收購工業公司，最終拿到數據但失去懂數據語義的人，剩下一堆無上下文的原始紀錄。

值得追問
  1. $62 億若分散到 6-8 家工業企業收購，每家整合複雜度都足以消耗一個管理團隊，Prometheus 的整合執行人才在哪裡？
  2. Bezos 真正的差異化資產是 AWS 的企業客戶關係，Prometheus 有沒有辦法把 AWS 通路轉化成工業 AI 的商業化入口？
  3. 如果 Prometheus 複製了 GE Digital 的失敗路徑，$62 億之後還有多少容錯空間？

Tether 要你在審計出爐前先把錢交出來

事實背景

Tether 正以 $5,000 億估值進行私人融資，計畫出售約 3% 股權募集 $150 至 $200 億，並設下 14 天投資者決定截止期。目前 USDT（Tether 發行的美元穩定幣）流通量為 $1,840 億，穩定幣市場總值達 $3,170 億。Tether 在 2026 年 3 月宣布聘請四大會計師事務所（Big Four：Deloitte、PwC、EY、KPMG 其中之一）進行公司史上首次完整審計（此前多年僅提供第三方認證），審計結果尚未出爐。

So What?

Tether 設 14 天截止期，但審計結果要數月後才出爐，投資者實際上是在「估值先鎖定、透明度後補齊」的條件下下注。這個賽局結構對現有股東極度有利：若審計乾淨，Tether 可以在 IPO 前以更高估值再融一輪，稀釋本輪投資者的意義；若審計揭露儲備品質問題，本輪投資者面臨的是無法退出的道德風險。$5,000 億對應的隱含倍數約為 38 倍年利潤，這個估值錨定的是今日利率環境，但若穩定幣監管（如 GENIUS Act）強制備付金分散，Tether 的套利空間將被系統性壓縮，本輪投資者將直接承擔利率和監管的雙重尾部風險。

值得追問
  1. 如果審計揭露儲備組成和外界預期有差距，14 天截止期是否讓投資者失去了重新議價的籌碼？
  2. GENIUS Act 若通過並要求穩定幣發行商分散備付金，$5,000 億估值的利潤假設會怎麼重算？
  3. Tether 的真實意圖是 IPO 前的最後一輪私募，還是借助「四大審計」敘事拉高估值後套現？

ETH 不是輸給 USDT，是輸給機構系統性出逃

事實背景

Polymarket 數據顯示，Ethereum 在 2026 年失去市值第二名的機率已從年初 17% 飆升至 60%。目前 ETH 市值約 $2,470 億，USDT 市值 $1,840 億，ETH 僅需再跌約 27%（至 $1,500 左右）即被翻轉，ETH 已從 2025 年 8 月歷史高點 $4,946 下跌超過 60%。Spot ETH ETF（直接持有 ETH 現貨的交易所交易基金，2024 年獲 SEC 批准）的 AUM（管理資產規模）從 2025 年 10 月的 $318.6 億暴跌至約 $117.6 億，蒸發 65%。

So What?

Spot ETH ETF 的 AUM 蒸發 65% 不是散戶恐慌，ETF 贖回需要機構主動操作，代表 BlackRock、Fidelity 等發行商的機構客戶正在系統性減持 ETH 敞口。ETH 失去 #2 地位本身對 DeFi 實際運作影響有限，但真正的受害者是 Optimism、Arbitrum 等 L2 基金會，這些組織大量持有 ETH 作為金庫儲備，敘事崩潰將同時打擊資產價值與募資能力。Tether 和 Circle 是這個對比中被強化的贏家，「功能性貨幣工具」取代「智能合約平台」佔市值第二，將是加密貨幣歷史上第一次。

值得追問
  1. 如果機構撤離 ETH 的邏輯是「缺乏明確的現金流故事」，Ethereum 基金會有什麼敘事可以重新吸引機構？
  2. L2 基金會的 ETH 金庫若被迫減持，會對 Arbitrum、Optimism 的生態激勵預算產生多大衝擊？
  3. 穩定幣市值超越智能合約平台，對整個 Web3 的「價值捕獲層在哪裡」這個問題，意味著什麼？

Eclipse 的閉環生態可能讓 LP 的退出估值打折

事實背景

Eclipse Ventures 於 2026 年 4 月 7 日宣布完成 $13 億新基金募資，為公司成立 11 年來最大單次募集，超越 2023 年 $12.3 億紀錄。資金分為兩個基金：Fund VI（$7.2 億，聚焦成長期）與 Early Growth Fund III（$5.91 億，聚焦早期孵化），管理資產規模達約 $100 億。投資組合包含電池回收公司 Redwood Materials、自動駕駛工地車輛 Bedrock Robotics、自駕技術公司 Wayve 與工業機器人實驗室 Mind Robotics。Eclipse 不僅投資，還在內部孵化新創，並主動推動投組公司互為客戶與合作夥伴。

So What?

Eclipse 的「投組互為客戶」模式對早期公司有真實好處，可以跳過冷啟動直接拿到 Redwood Materials 或 Wayve 的採購合約。但這個模式對 LP 是一把雙面刃：若生態系內的關聯方交易佔各公司營收比例過高，退出時 PE 買家或戰略收購方會把這部分視為不可持續收入，直接折扣估值。Eclipse 在用「產業系統整合者」的邏輯做 VC，但如果它的被投公司最終只能在 Eclipse 生態內存活，外部退出的流動性就會被嚴重限制。

值得追問
  1. 如果 Bedrock Robotics 的客戶有 60% 來自 Eclipse 投組，這家公司的獨立市場定價能力要怎麼評估？
  2. Eclipse 的孵化和投資基金分開設立，利益衝突怎麼管？LP 的出資在兩個基金間的分配邏輯是什麼？
  3. 「物理 AI」的退出路徑是 IPO 還是被 GE、Honeywell、Lockheed 這類產業巨頭收購？這兩條路對估值的含義差很遠。

Dragonfly 轉向「現實金融」，DeFi 創辦人的融資窗口正在關閉

事實背景

加密原生 VC Dragonfly 於 2026 年 2 月完成 Fund IV 募集，規模 $6.5 億，超出原定 $5 億目標，管理資產規模約 $40 億。投資主題從純 DeFi 協議轉向穩定幣、去中心化預測市場、RWA（Real World Assets，將房地產、國債等現實資產代幣化後放上鏈交易）、類信用卡服務與貨幣市場基金。前一期 Fund III 代表性投資包括 Polymarket、Rain 與 Ethena。鏈上數據顯示，Dragonfly 於 4 月 5 日將 $212 萬 MNT（Mantle Network 原生代幣）轉入 Bybit，為 2 個月來首次資金流出。

So What?

Dragonfly 的主題轉向不是前瞻性的主動選擇，而是 DeFi 協議代幣普遍跌 80% 以上之後被迫重新定義「可退出資產」的範疇。贏家是已在 Fund III 佈局的 Ethena 和 Polymarket，頭部加密 VC 的敘事升級直接為這些標的帶來估值支撐。輸家是純鏈上 DeFi 協議的創辦人，當 Dragonfly 這個加密原生 VC 的旗手都把資金轉向更接近傳統金融的資產，這個類別的融資窗口正在系統性收窄，不是暫時的市場低潮。

值得追問
  1. 如果頭部加密 VC 都在轉向 RWA 和穩定幣基建，下一個週期還會有人給純鏈上 DeFi 創新提供早期資金嗎？
  2. Dragonfly 的鏈上 $212 萬 MNT 流出發生在 Fund IV 剛完成募集之後，這是投組再平衡信號還是有更多後續操作？
  3. RWA 代幣化需要監管許可，這和 DeFi「無需許可」的核心敘事是根本衝突，Dragonfly 怎麼跟 LP 解釋這個策略轉身的邏輯自洽性？

Anthropic 最強模型不公開賣，反而成了它最有力的競爭武器

! Mythos 的「危險性」完全是 Anthropic 自我認證，沒有第三方能驗證，這變成了一個不可證偽的論述。
事實背景

Anthropic 在 2026-04-17 同日發布兩件事：一是 Claude Opus 4.7 全球正式上線（Anthropic API、Amazon Bedrock、Microsoft、Google Cloud），定價 $5/$25 per M tokens 與前代相同，SWE-bench Verified（軟體工程能力評估基準）得分從 80.8% 升至 87.6%，SWE-bench Pro 從 53.4% 跳到 64.3%，全面領先 GPT-5.4 的 57.7% 和 Gemini 3.1 Pro 的 54.2%。二是 Bloomberg 同日披露 Mythos Preview 被限制在 Project Glasswing 內僅 40 家機構使用，原因是模型可自主串聯 Linux kernel 漏洞取得完整控制權，SWE-bench Verified 達 93.9%、GPQA Diamond 達 94.6%，並在每一款主流作業系統與瀏覽器找到「數千個」零日漏洞（zero-day，尚未公開的系統安全漏洞）。

So What?

Mythos 的整套「危險性」敘事完全依賴 Anthropic 自有紅隊（red-team），Project Glasswing 的 40 家機構沒有獨立重現義務，NIST AI Safety Institute 等第三方機構未被授權公開驗證。Anthropic 在這件事上同時是裁判和選手，不可證偽的安全宣稱讓這個決策的真實動機，無論是安全還是競爭策略，外部無法區分。對 OpenAI 和 Google 而言，若日後宣告同樣的「hold-back」，必須拿出等量的危險能力背書，否則就是承認自己能力不足，Anthropic 已經把 AI 發布節奏從一維（快慢）改寫成二維（快慢加上能不能公開）。

值得追問
  1. 如果第三方機構明天被允許驗證 Mythos，而結果顯示漏洞嚴重性被誇大了，Anthropic 的安全論述會不會反過來成為公關危機？
  2. Glasswing 成員能不能把 Mythos 掃出的漏洞情報用在自己的商業安全產品上？如果可以，CrowdStrike 和沒入選的 Tenable、Rapid7 之間的能力差距會有多大？

事實背景

Anthropic 年化營收（ARR，Annual Recurring Revenue）從 2025-01 的 $1B，在 15 個月內增長至 2026-04 的 $30B，超越 OpenAI 的 $25B。Series G 結束後不到兩個月，年度花費超過 $1M 的企業客戶數從 500 翻倍到 1,000；企業客戶佔總營收 80%，相比之下 OpenAI 偏消費端。Anthropic 訓練支出約為 OpenAI 的 1/4。公司目標 2026-10 IPO，屆時私募估值 $380B，計畫募資 $60B。

So What?

OpenAI 把 agent 執行環境做進 SDK，真正要防的是自己被降格成 token 供應商

事實背景

OpenAI 於 2026-04-15 宣布 Agents SDK 新增 model-native harness（模型原生執行殼，讓 agent 可跨檔案和工具操作）與原生 sandbox 執行環境（受限隔離計算環境）。Codex 同步新增 in-app browser，可在渲染頁面上直接標注讓 Codex 回應。首發 Python，TypeScript 後續跟進。使用標準 API 定價，無額外費用。

So What?

企業把 agent 狀態管理、sandbox 日誌、錯誤恢復邏輯寫進 OpenAI SDK 之後，跨廠商遷移的工程成本會遠高於表面上的工具定義重新封裝，因為兩套系統（OpenAI harness 和 Anthropic MCP）雖然都是 JSON-based tool-calling，互斥性低，但 runtime 層的深度整合才是真正的鎖定點。OpenAI 急發這個 SDK 的真實驅動力，不是擔心 MCP 成為協定標準，而是擔心 Cursor 和 Replit 等第三方把 agent 執行層吃掉，讓 OpenAI 退化為純 token 供應商。

值得追問

  2. Anthropic 的目標是統一 agent-工具協定，OpenAI 的目標是掌控 agent 執行環境，這兩個策略能共存多久？

Solana 週轉速度可能被誇大 30-50%，但即使如此仍在贏

事實背景

Solana 穩定幣總供應達 $16.288B，7 天增加 $1.6B（+10.87%），為前 10 公鏈中增速最快。截至 2026-04-12 的兩週，Solana 以 32.6% 佔據調整後（剔除 wash trading 和 MEV 機器人交易後）美元穩定幣週成交量第一，超越 Ethereum 的 27.8%。每單位 USDT 的週轉速度（velocity，每單位穩定幣的時間內轉手次數），Solana 約為 Ethereum 的 6 倍。Solana Q1 2026 總經濟活動 $1.1T，環比增長 6,558.6%；Ethereum 穩定幣供應仍佔全球約 60% 份額（$180B）。

GENIUS Act 的境外漏洞不是立法失敗，是刻意為 Tether 留的門

事實背景

FDIC Board 於 2026-04-07 批准 GENIUS Act（美國穩定幣監管法案，2025-07-18 簽署生效）實施規則草案，要求穩定幣發行者持有 1:1 高品質流動資產（現金或短期美債）儲備、月度披露、48 小時內以面額贖回。草案於 2026-04-10 在 Federal Register 刊登。境外發行者如 Tether 可在美國市場流通而無需提供完整儲備披露。全球穩定幣市值目前超過 $200B。Reed 參議員 2026-02-27 提案修補境外漏洞，尚未推進。

So What?

GENIUS Act 框架下，Circle 承擔全部合規成本，卻面對一個不受同等披露限制的競爭對手。但 Circle 的真實補償是 Tether 永遠無法複製的：Fed Fedwire 即時結算接入、銀行同業 USDC 抵押品認可、FedNow 整合，這些通路不依賴 Reed 修正案是否通過，是 GENIUS Act 在機構場景中對 Circle 的結構性補償。真正被碾壓的是 Paxos 等中型合規發行者，它們承擔合規成本，卻既沒有 Circle 的銀行通路深度，也沒有 Tether 的境外規避空間。

  3. Reed 修正案的停滯，是 Tether 遊說的結果，還是沒有足夠選票推進的技術問題？

Samani 退出 Multicoin 的結構，可能是下一批 crypto GP 學習的範本

事實背景

Kyle Samani 於 X 宣布，2026-03-31 以 in-kind（實物贖回，不以現金而以資產形式交換）方式退出 Multicoin，獲得 Forward Industries（FWDI，一家以 SOL 為主要財務儲備的上市公司）股票和認股權證（warrants），而非現金。Samani 仍擔任 FWDI Chairman 和 Zama 董事，自陳「mega long SOL、mega long crypto」，將以個人身份繼續投資並支持 Multicoin portfolio 公司。Tushar Jain 留任 Multicoin GP。

So What?

Samani 的 in-kind 贖回本質上是把個人風險敞口從「受基金存續期限制的 SOL 曝險」轉換為「FWDI 這類永久資本載體的 SOL 曝險」，他退出的是 Multicoin 的鎖定期和 LP 問責結構，不是 SOL 本身。贏家是 Samani 個人，保留上漲空間且移除了基金期限約束；輸家是 Multicoin 現有 LP，流動性提供者縮減，未來 NAV 支撐力下降。這個模式要複製需要三個前提：有願意以股票換 fund 持倉的上市 treasury 公司、GP 退出不觸發 LP 協議違約、市場對 treasury 公司股票有足夠流動性溢價，目前只有 FWDI 這類小型 SOL treasury 公司同時達標。

值得追問
  1. 如果這個模式開始被複製，傳統 crypto fund 的 LP 協議會怎麼修訂來防止 GP 以 in-kind 方式繞過鎖定期？
  2. FWDI 作為 SOL treasury 公司的股票流動性，足夠讓這個模式在更大規模的 fund 上複製嗎？

a16z 的穩定幣「本地化」數據，贏家恰好都是 a16z 自己的被投公司

! 報告引用的增長數字幾乎無法獨立驗證，最可能的解釋是 DeFi 自循環而非真實商業支付。
事實背景

a16z crypto 於 2026 年 4 月 25 日發布一份附有 9 張圖表的報告，梳理穩定幣市場結構演變。核心數字：Q1 2026 穩定幣調整後交易量達 4.5 兆美元；穩定幣 velocity（轉手率，衡量每單位供應被使用的頻率）從 2024 年初的 2.6 倍升至 6 倍；C2B（消費者對商家）交易 2025 年同比增長 128%，達 2.846 億筆；境內交易佔比從 2024 年初的約 50% 升至 2026 年初的近 75%。巴西雷亞爾穩定幣 BRLA 整合 PIX（巴西央行即時支付網絡）後，月交易量從近零成長至約 4 億美元。Rain 平台穩定幣卡月抵押品存入同期從接近零超過 3 億美元。

So What?

報告引用的 3,500 億到 5,500 億美元「支付流量」，以及 Rain 和 BRLA 的具體增長數字，絕大概率來自 a16z portfolio 公司自報，而非第三方鏈上去重數據。全文沒有引用任何 Nansen 或 Dune Analytics 的獨立交叉驗證，技術調整剔除標準也未公開。velocity 從 2.6 倍翻倍至 6 倍，與境內佔比同步上升，最合理的解釋是 DeFi 協議之間的 composability 循環（同一筆穩定幣在 Aave、Uniswap、Curve 間反覆調用均計為境內交易），而非真實商業支付增長。

值得追問
  1. 如果把「境內交易」的定義換成「鏈上非 DeFi 協議間轉帳」，境內佔比還會是 75% 嗎？
  2. BRLA 的護城河建立在 PIX 整合上，但 PIX 是巴西央行的基礎設施。如果巴西央行要求穩定幣發行商持有本地牌照並以國債做準備金，現有 BRLA 投資人的商業模式還剩什麼？
  3. 一份研究報告的發布方同時持有報告中大部分數據來源公司的股份，你會怎麼打折這份報告的可信度？
CoinDesk Web3

x402 拉來了 Microsoft、Google、Mastercard，但 4,800 萬美元裡幾乎沒有他們的錢

事實背景

Coinbase 的 Base（Ethereum Layer-2 網絡）開發負責人 Jesse Pollak 接受 CoinDesk 採訪，推廣 x402 開源支付協議。x402 允許 AI agent（自主執行任務的 AI 程序）透過單次 API call 完成即時支付，幾乎零手續費。截至發布，x402 累計處理 4,800 萬美元支付量，95% 發生在 Base。Coinbase 同步推出 Agentic.Market，一個面向 AI agent 的應用商店。Microsoft、Google、Mastercard 被列為協議合作夥伴。

So What?

4,800 萬美元的累計量若由 Microsoft、Google、Mastercard 三家 production 級別使用所貢獻，每家平均不到 1,600 萬美元，對這三家的業務規模而言是噪音，這幾乎確認他們目前只是 working group 層級的開源貢獻者，而非商業整合夥伴。x402 的真正結構性缺陷在別處：鏈上不可逆結算放棄了傳統 interchange fee 的 chargeback 基礎設施，一旦涉及訂閱、SLA 保證或部分交付，這個缺口就是致命的。真正的受益者是 Stripe 和 PayPal，只要他們推出 API-native 的 agent 付費端點，退款保護就是決定性的差異化。

值得追問
  1. AI agent 自主付款的場景裡，「付錯了但無法退款」這個風險，應該由誰承擔？開發者、用戶，還是協議本身？
  2. 如果 Stripe 明天推出一個帶 chargeback 保護的 agent 支付 API，x402 的價值主張還剩什麼？
  3. 「加密大眾採用」如果發生在用戶完全感知不到區塊鏈的場景，這對公鏈代幣的估值邏輯意味著什麼？

TechCrunch / MIT Technology Review AI

DeepSeek V4 的真正威脅不是技術趕超，而是它正在強迫 OpenAI 重新定價

! DeepSeek 的低價定位正侵蝕 Anthropic 的企業 ARR 增速假設，這對後者的高估值結構是慢性威脅。
事實背景

DeepSeek 於 2026 年 4 月 24 日發布 V4 preview，推出兩款 API 模型。V4-Pro 採用 MoE（Mixture-of-Experts，總參數大但每次推理只激活部分的模型架構）架構，總參數 1.6 兆，每次推理激活 49B 參數；V4-Flash 總參數 284B，激活 13B。兩款均支援 1M token context window 以及 Thinking（模型在回答前進行明確推理鏈）和 Non-Thinking 雙模式。定價上，V4-Pro 為每百萬 tokens 輸入 1.74 美元、輸出 3.48 美元；V4-Flash 為 0.14 美元輸入、0.28 美元輸出。TechCrunch 與 MIT Technology Review 均指出 V4 在推理 benchmark 上已幾乎關閉與前沿模型的差距。

So What?

DeepSeek 的 1.74 美元定價是否覆蓋真實算力成本，在沒有任何公開財務披露的情況下無法確認，幻方量化的交叉補貼目前只是最可信的推測。但定價壓力本身是真實的，不論成本結構如何，接近前沿性能加上數倍的價差，已經在侵蝕 OpenAI 和 Anthropic 的 API 定價空間。Anthropic 最脆弱的部分不是企業客戶流失，而是企業開始把非敏感工作負載切給 DeepSeek 做成本套利，讓 Anthropic 高估值依賴的 ARR 增速假設難以維持；DeepSeek V4 目前仍在 preview 階段且缺乏企業級 SLA 和數據主權協議，短期內歐美受監管企業的 production 部署可能性接近零。

值得追問
  1. 如果 DeepSeek 的低價是交叉補貼而非真實成本，幻方量化什麼情況下會停止輸血？到那時市場格局會長什麼樣？
  2. 企業把非敏感工作負載切給 DeepSeek、敏感工作負載留在 Anthropic，這個分層格局對 Anthropic 的估值邏輯是救命稻草還是慢性毒藥？
  3. AWS 和 Google Cloud 同時分發 Anthropic 和 DeepSeek，對他們來說 AI 能力商品化是好事。這個格局最終對誰最有利？
